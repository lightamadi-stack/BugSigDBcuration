# BugSigDB-Curation Portfolio

This repository contains my curated microbial signatures from research articles for the BugSigDB project.  
Each folder represents one curated paper, including:

- Study description  
- Study design  
- Exposure and control groups  
- Statistical tests used  
- Significant taxa (increased and decreased)  
- Alpha diversity results  
- NCBI taxonomy IDs  
- Notes on curation decisions  

All curations follow the official BugSigDB curation policy.

## 📚 Repository Structure

Each folder includes a `README.md` file with:
- Full paper citation  
- Summary of the study  
- Extracted microbial signatures  
- Details of the curation process  

## 🧩 Purpose of This Repository

This repository serves as:
- A portfolio of my microbiome curation work  
- Documentation of my contributions to BugSigDB  
- A reference for future microbiome-related projects  
- Evidence of open-source contributions
