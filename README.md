# BugSigDB-Curation Portfolio

Welcome to my BugSigDB Curation Portfolio!
Here, I’m actively curating microbiome studies for the BugSigDB project and documenting each step, from extracting microbial signatures to interpreting study designs and standardizing taxa information.

## 🧩 Purpose of This Repository

This repository serves as:
- A portfolio of my microbiome curation work  
- Documentation of my contributions to BugSigDB  
- A reference for future microbiome-related projects  
- [Record of open-source contributions](https://bugsigdb.org/Special:Contributions/LightAmadi)  

---

## 🧠 My Role in BugSigDB

I contribute by:

- Curating microbial signatures from published research articles
- Interpreting statistical outputs (LEfSe, Wilcoxon, Mann–Whitney, DESeq2, Random Forest, Poisson, ANOVA, Songbird analysis, SparCc, etc.)
- Identifying valid taxa using the NCBI Taxonomy Browser
- Applying BugSigDB curation rules to ensure consistency and accuracy
- Documenting study design, matching, confounders, and significant taxa
- Engaging in team-based collaboration
